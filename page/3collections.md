## Reverse Engineering Tools

* [IDA Pro](https://hex-rays.com/ida-pro/)

    The industry standard disassembler and debugger for binary analysis.

* [Ghidra](https://ghidra-sre.org/)

    NSA's open-source software reverse engineering framework.

* [Binary Ninja](https://binary.ninja/)

    Modern disassembler with a powerful API for automation.

* [Radare2](https://rada.re/n/)

    Open-source framework for reverse engineering and binary analysis.

* [x64dbg](https://x64dbg.com/)

    Open-source x64/x32 debugger for Windows.

* [Hopper Disassembler](https://www.hopperapp.com/)

    Reverse engineering tool for macOS and Linux.
